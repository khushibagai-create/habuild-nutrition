Home — empty state
Brand-new member. Your recipes shows the "Save what you cook" empty card with green CTA button.

Today's recipe detail
Opened from cover or "Read recipe →". Hero, voice tip, ingredients, method, benefits, Q&A, share.

Community + Own recipes
Screen 04
Member detail — community
Someone else's recipe. Heart pill on hero. No Edit/Delete pills.
Scroll for ingredients · method · more member dishes
Screen 05
Member detail — own
Member's own recipe. ✎ Edit recipe + 🗑 Delete pills visible below dish title.
Scroll to see Edit + Delete pills
Add · Edit · Delete
Screen 06
Add recipe — empty
Photo zone · Name · How to make · Why it's special. Submit disabled until all 3 text fields filled.
Screen 07
Add recipe — filled
All fields complete. Submit button is forest-green and active: "Share with the community".
Screen 08
Recipe submitted — success
Lands here after submit. WhatsApp share (primary) + "Done — back to home" text link (secondary).
Screen 09
Edit recipe — pre-filled
Triggered by ✎ pill. All fields pre-populated. Submit becomes "Save changes".
Screen 10
Recipe updated — success
Same layout as Screen 08 with "Recipe updated" / "Your changes are live for the community."
Screen 11
Browse all recipes
Meal-time chip (primary) + Condition pill (secondary). Tap any tile → recipe detail.
Long scroll — full recipe library inside
